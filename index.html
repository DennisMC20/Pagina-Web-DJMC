¿ Que es HTML5?

Esto fue posteado el 10/07/2018/ por UMG-Jalapa

Durante los ultimos años, el estandar HTML5 se ha ido ganando una importancia creciente, y que ahora pasa a ser la protagonista indiscutible después del fin de soporte de Flash Player.

HTML5 es un lenguaje mas semantico. En otras palabras es procesado por los navegadores web regulares así como por muchos otros agentes de usuarios. Las CSS son usadas para sugerir la presentación de la página a usuarios humanos.

Introduccion a CSS3

Esto fue posteado el 10/07/2018/ por UMG-Jalapa

El nombre hojas de estilo en cascada viene del ingles Cascading Style Sheets, del que toma sus siglas. CSS es un lenguaje para definir la presentación de un documento estructurado escrito en HTML o XML.

para que sirve el CSS3?

CSS es un lenguaje de diseño gráfico que permite definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer el diseño visual de los documentos web e interfaces de usuario escritas en HTML.
